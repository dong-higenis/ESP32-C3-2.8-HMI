## 기울기 센서 GUI

### 프로젝트 개요
이 프로젝트는 TFT_eSPI 라이브러리를 사용하여 2축 수평계 센서(X/Y 축 경사 센서)의 데이터를 실시간으로 시각화하는 Arduino 기반 임베디드 시스템입니다. 
320x240 해상도의 TFT LCD 디스플레이에 센서 값을 그래픽 인터페이스로 표시하며, 프레임버퍼 기반 렌더링을 통해 화면 깜빡임을 최소화합니다.

### NOTE (테스트 환경)
esp32 by Espressif Systems - ver 2.0.10

### 주요 기능
1. **`원형 HUD 디스플레이`**

- 3중 원 구조 (큰 원: 최대 범위 / 중간 원: 평행 기준 / 작은 원: 현재 위치 포인터)

- 십자 기준선으로 수평/수직 가이드 제공

- 벡터 기반 방향 계산으로 직관적인 기울기 표현

2. **`축별 가이드 바`**

- X축(가로) / Y축(세로) 독립 표시

- 타원형 포인터와 6등분 눈금선

- 실시간 offset 값 반영

3. **`수치 데이터 표시`**

- 소수점 둘째 자리까지 정밀 표시

- ±부호 포함 가독성 향상

### 기술적 특징

프레임버퍼 렌더링

모든 그래픽 요소를 RAM에서 합성 후 한 번에 전송하여 화면 깜빡임 제거

비선형 좌표 매핑

0~0.11도(평행 구간)는 중간 원 내부에, 그 이상은 큰 원까지 확장 매핑하여 미세한 기울기 변화 감지


파일 구조
├── ui_draw.cpp      # 메인 렌더링 로직

├── ui_draw.h        # 공개 API

├── ui_def.h         # UI 상수 (좌표/색상/크기)

├── font.h           # 커스텀 5×7 비트맵 폰트

└── main.ino         # Arduino 메인 루프

### 하드웨어 요구사항

**`MCU`**: ESP32C3

**`디스플레이`** : 320×240 TFT (ILI9341/ST7789)

**`기울기 센서`** : TiltSense (by Higenis)

**`통신`** : 기울기 센서와 UART로 통신 합니다.
